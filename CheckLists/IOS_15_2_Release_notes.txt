IOS 15.2 release notes

1. Представлена функция записи активности приложений в настройках конфиденциальности, позволяющая людям сохранять сводку датчиков, данных и доступа в Интернет для приложений на своих устройствах. Это действие представлено в: «Настройки»> «Конфиденциальность»> «Отчет о конфиденциальности приложений». 
2. API StoreKit, которые представляют лист запроса на возврат, можно протестировать с помощью StoreKit Testing в Xcode.
3. API StoreKit, которые представляют лист в приложении для управления подписками, можно протестировать с помощью StoreKit Testing в Xcode. 
4. Новые значения SKTestSession.TimeRate доступны для использования в автоматических тестах в StoreKit Test framework. (82680742).
5. Были добавлены Legacy Contacts в iOS и iPadOS 15.2, что позволяет пользователям назначать людей в качестве старых контактов для своих учетных записей в рамках программы Digital Legacy.
6. Подписчики iCloud + теперь могут получить доступ и использовать функцию: «Скрыть мою электронную почту» прямо из приложения «Почта».
7. Автоматический вызов теперь можно настроить на использование одного из двух методов вызова службы экстренной помощи: удерживание боковой кнопки вместе с кнопкой громкости или быстрое нажатие боковой кнопки несколько раз. Оба метода теперь показывают более длительный 8-секундный обратный отсчет перед вызовом службы экстренной помощи.
8. Теги теперь можно массово переименовывать и удалять.
9. Core Media. Known issues:
Потоковая передача в приложении Музыка может привести к более высокой загрузке ЦП, что в некоторых случаях приведет к более быстрой разрядке батареи.
10. SwiftUI 
Resolved issues:
- Нажатие ScrollView, к которому применен фон, внутри стиля стека NavigationView, когда внутри TabView, теперь правильно отслеживается с помощью navigationBar и tabBar. (83686857)
- Использование alert (_: isPresent: actions: message :) и confirmDialog (_: isPresent: titleVisibility: actions :) теперь присутствует. (83731075)
- List правильно учитывает вставки Safe area. (83312573)
- Views больше не скрываются при использовании указателя iPad для отображения контекстного меню. (83953549)
	Known issues:
- TextField не будет пытаться локализовать необработанную строку TextField ("Placeholder", text: $text) при построении для iOS 15 и запуске в более старых версиях.

- Обходной путь: заключите строку в LocalizedStringKey. (82076857)
